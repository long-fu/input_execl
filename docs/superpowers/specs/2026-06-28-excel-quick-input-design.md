# Excel 快速录入工具 — 设计文档

日期：2026-06-28

## 概述

基于 Python + tkinter 的桌面应用，支持通过键盘快速向 Excel(.xlsx) 表格中输入数据。核心交互：输入列号 → 行号 → 数据值 → 回车，数据写入对应单元格。

## 项目结构

```
input_execl/
├── main.py              # 入口
├── ui/
│   ├── __init__.py
│   ├── app.py           # 主窗口，组装布局、菜单、事件绑定

├── core/

│   └── navigator.py     # 录入模式逻辑
└── requirements.txt     # openpyxl
```

## 运行时录入流程

1. 用户在输入栏依次填写列号、行号、数据值
2. 按回车触发录入
3. 系统解析列号(支持字母 A-ZZ 和数字)、行号，写入 Excel
4. 刷新表格视图，高亮刚写入的单元格
5. 根据当前模式更新列/行位置，焦点回到数值框

## 三种模式

- **逐格录入**：写入后清空数值框，不自动递增
- **按列递增**：写入后行号+1
- **按行递增**：写入后列号+1

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Tab | 输入框间切换 |
| Enter | 在数值框时触发录入 |
| Ctrl+O | 打开文件 |
| Ctrl+S | 保存 |
| Ctrl+N | 新建 |
| Ctrl+1/2/3 | 切换模式 |
| Escape | 清空输入 |

## 点击交互

点击 Treeview 中单元格 → 自动填充列号、行号到输入栏 → 焦点跳到数值框。

## 依赖

- Python 3.8+
- tkinter（标准库）
