# 警戒列表面板设计

## 概述

右侧面板拆分为上下两部分：上方为输入记录（60%），下方为警戒列表（40%），展示固定行中超过警戒值的单元格。

## 功能要点

- 输入记录面板高度占 60%，警戒列表占 40%
- 仅固定行模式下显示警戒内容
- 每行格式：`列号 × 行号 = 值`（如 `3 × 5 = 150`），列号使用数字
- 表格列标红保留，与警戒列表共存
- 刷新时机：提交、撤销、清空行、模式切换

## 组件改动

### App (`ui/app.py`)

- 右侧 `log_frame` 拆分为上下两个子 Frame
- 上方保持现有输入记录 Text
- 下方新增警戒列表 Text（只读）
- 新增 `_update_alert_list()` 方法

### InputBar / TableView

- 不动
